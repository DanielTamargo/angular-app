ANGULAR



App developed in order to learn everything about Angular


Technologies used


I wanted to start developing my own projects as web developer in order to get better.
I realized that in Spain tons (literally, tons) of companies are looking for frontend developers and Angular was one of the most requested.

As I said, my objective was to leave my comfort zone and try new things. So... What did I use?

Framework

Angular was my choice and I tried to use all its features like angular animations, services, guards, canActivate, canDeactivate, custom pipes, routes, modules...
Also it was a perfect oportunity to learn more TypeScript.

App State
  • RxJS: observables, observers, operators... hard at the beggining, loved it at the end!
  • NgRx: wanted to learn about this Angular's Redux version so I decided to use it in one of the three mini-apps
Style
  • SASS: applying cool styles was not the objective here but Sass is such a powerful way to develop CSS
  • Bootstrap: fast styling with this classic toolkit
  • Angular Material: best way to have cool visual and useful components already built
Libraries
  • Open Layers: loading map and its layers (awesome library! an alternative would be Leaflet)
  • AngularFire: to implement firebase with Angular in a easy way
  • FirebaseUI-Angular: fast login screen for firebase
APIs
  • GitHub: open API used to scout profile's surface
  • OpenDataSoft: used to load GeoJSONs as feature layers for the maps
  • Bored API: are you bored? this API gives you random activities to do!

APPs inside this project

You can find 3 mini-apps inside this project

GitHub Profile Scouter

Use this mini-app to check user profile's basic info, repos, gists, followers and followings

You can display repository's info and check how many commits it have and who were the contributors to the repo.

Tasklist

The classic TODO mini-app. You can create tasks, delete or mark them as completed or even get a random one.

You have to log in in order to use this app, so you can check your tasks from any device!

Map

Wanna know which autonomous communities are in Spain?

Go ahead and use this mini app! Use the panel and load any autonomous cumminity (or go big and load them all!)

This app was a great way to learn basics about GIS (like WMS, WMTS, OSM, EPG projections, GeoJSON...)
